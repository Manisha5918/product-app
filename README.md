# Product App - React E-Commerce Application

A modern responsive E-Commerce web application built using **React.js**.  
The application allows users to explore products, view product details, manage cart and wishlist items with a clean user-friendly interface.


##  Features

###  Authentication
- User Login System
- Protected Routes
- Persistent Login State using Local Storage


###  Home Page
- Modern landing page
- Product introduction section
- Responsive UI design


### Product Management
- Fetch products from FakeStore API
- Display product listings
- Detailed product information page
- Product images
- Category and price details


###  User Features
- Search products
- Filter products
- Add products to cart
- Remove products from cart
- Wishlist management
- User profile page


###  UI Features
- Dark modern theme
- Responsive layout
- Interactive buttons
- Toast notifications
- Smooth navigation



## Technologies Used

- ⚛️ React.js
- JavaScript ES6
- HTML5
- CSS3
- React Router DOM
- Axios
- Local Storage
- FakeStore API

##  Installation

Clone the repository

```bash
git clone https://github.com/your-username/product-app.git
```

Open project folder

```bash
cd product-app
```

Install dependencies

```bash
npm install
```

Run application

```bash
npm start
```
