# Product App

A React E-Commerce Product Application using FakeStore API.  
The application allows users to browse products with a clean and responsive interface.

## Features

- View product listings
- Product details display
- Search products
- Filter products
- Shopping cart functionality
- Wishlist management
- CRUD operations

## Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3
- Axios
- FakeStore API
